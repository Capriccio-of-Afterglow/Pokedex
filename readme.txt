功能(API):

使用者-User:
1. 查詢所有使用者(多筆)
2. 新增使用者
3. 修改密碼
4. 根據使用者 ID 修改使用權限等級
5. 根據使用者名稱查找使用者(登入用-單筆)
6. 根據使用者ID查找使用者(單筆)

寶可夢-pokemon
1. 查詢所有寶可夢(多筆)
2. 新增寶可夢
3. 根據寶可夢名稱查找寶可夢(單筆)
4. 根據寶可夢ID查找寶可夢(單筆)
5. 修改寶可夢資料
6. 根據ID刪除寶可夢

背包(Package)
1.查詢所有背包內的寶可夢
2.透過使用者ID查詢使用者背包內的寶可夢
3.新增寶可夢到使用者背包中
4.修改背包內寶可夢的CP值
