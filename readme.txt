功能(API):

使用者(user)
1. 查詢所有使用者(多筆)
2. 新增使用者
3. 修改密碼
4. 根據使用者 ID 修改使用權限等級
5. 根據使用者名稱查找使用者(登入用-單筆)
6. 根據使用者ID查找使用者(單筆)


寶可夢(pokemon)
1. 查詢所有寶可夢(多筆)
2. 新增寶可夢
3. 根據寶可夢名稱查找寶可夢(單筆)
4. 根據寶可夢ID查找寶可夢(單筆)
5. 修改寶可夢資料
6. 根據ID刪除寶可夢

寶貝球(pokeball)
1.查詢所有的寶可夢球(多筆)
2.查詢使用者的寶可夢球(多筆)
3.新增寶可夢到寶可夢球中
4.修改寶可夢球中的寶可夢的CP值
5.根據寶可夢ID查詢寶可夢球(用於查詢自己的寶可夢強度排行)
6.根據使用者ID及寶可夢ID查詢到寶可夢(多筆)
7.根據寶可夢球ID找到使用者名稱及寶可夢名稱(單筆)
8.根據寶可夢球ID刪除寶可夢

(保留)
8.跟其他使用者交換持有的寶可夢
  8-1.邀請交換者發出交換請求(發出者先選擇要送出的寶可夢)
  8-2.被邀請者選擇接受或拒絕(拒絕則取消交換)
  8-3.被邀請者選擇要送出的寶可夢，雙方交換。

屬性(type)
1.列出所有屬性(多筆)

屬性關聯表(pokemon_type)
1.根據屬性ID列出該屬性的寶可夢(多筆)
2.根據寶可夢ID列出該寶可夢的屬性(多筆)
